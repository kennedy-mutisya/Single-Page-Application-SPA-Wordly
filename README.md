Dictionary SPA Lab

A Single Page Application (SPA)** built with HTML, CSS, and JavaScript** that lets users search for words, view definitions, synonyms, pronunciation, and play audio when available. The app also stores search history using localStorage, making it a practical teaching resource for modern JavaScript development.
